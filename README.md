## AIM
This project aims to simplify the vehicle rental process, offering users a hassle-free experience from browsing available vehicles to making secure online payments. With features such as detailed vehicle listings, advanced filtering options, and convenient booking management, users can find the perfect vehicle to suit their needs with ease. Additionally, the system provides administrators with insightful analytics and efficient tools for managing bookings, ensuring smooth operations and enhanced customer satisfaction.

- **XAMPP Control Panel**
- **PHP**
- **SQL**
